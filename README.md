# 数字丝绸之路在地化 · Localizing the Digital Silk Road

> An interactive HTML presentation built for the 清华大学首届区域国别研究博士生论坛 (Tsinghua Inaugural Doctoral Forum on Area Studies, 2026)

**作者** · 刘宇轩 Hakim Liu · Department of Malay Studies, National University of Singapore
**研究场域** · 马来西亚 · 槟城（2022.04 – 2024.04，参与式观察）

## About

A zero-dependency interactive web presentation that traces how China's Digital Silk Road actually lands in a single state of Malaysia. Based on two years of participant observation as a regional manager of a Chinese tech enterprise in Penang.

**18 slides** covering:
1. Research question · the gap in DSR literature
2. Interactive map of the fieldsite (Leaflet + halftone overlay + 5 stylized zones)
3. The "three-fold embeddedness" framework (institutional · network · knowledge)
4. H Company organizational architecture (the single Malay employee, visualized)
5. Detailed salary-gap comparison (Chinese expat vs local employee · 3–5×)
6. Bumiputera / ePerolehan / DBKL — institutional friction
7. PCCC and the 30-year continuity of Chinese commercial networks
8. Language as a trust boundary (English → Bahasa → Mandarin → Hokkien)
9. **Interactive Batu Kawan field** — click US/Chinese/Malaysian actors to reveal the supply / membership ties that make the China–US tech rivalry dissolve at the factory floor
10. Knowledge translation chain + multi-ethnic time rhythms
11. Findings + radar comparison
12. Synthesis · methodological reflection · closing references map

## Features

- **Single static HTML** · no build step, no npm dependencies at runtime
- **Leaflet + CARTO basemap** with custom halftone overlay & 5 styled zone polygons
- **3D mouse-tilt** framework triangle (Slide 7)
- **Per-character mouse-repulsion** on the title (Slide 1)
- **Interactive supply-chain graph** for Batu Kawan industrial park (Slide 13)
- Hemicycle visualization of Penang 2023 state election (Slide 6)
- Full keyboard / touch navigation, smooth scroll snap, prefers-reduced-motion support

## Run locally

Just open `index.html` in any modern browser. No build needed.

```
# Or serve via any static server
npx http-server .
```
